# Contributing

Thank you for your interest in contributing to Rsme. Here's how you can help.

## Getting Started

1. Fork the repository
2. Create a new branch for your feature or fix
3. Make your changes
4. Run the linter: `npm run lint`
5. Submit a pull request

## Development

- The frontend is a React + TypeScript app served by Vite on port 3000
- The backend is an Express server for document parsing on port 3001
- Run both concurrently with `npm run dev`

## Code Style

- TypeScript with strict mode enabled
- Components use functional patterns with hooks
- Tailwind CSS for styling

## Pull Request Guidelines

- Keep PRs focused on a single concern
- Write clear commit messages
- Update the README if your change adds or modifies a feature
- Ensure the linter passes (`npm run lint`)
